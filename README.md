# UAE E-Commerce Shopper Insights — Noon vs Amazon.ae

## What this project is
Sentiment analysis of 4,000+ product reviews from Noon and Amazon.ae,
combined with a survey of 70 UAE-based online shoppers. Identifies what
drives positive and negative experiences across electronics, home
appliances, and beauty categories.
